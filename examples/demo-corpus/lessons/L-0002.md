# CI green proved nothing — the suite silently skipped every integration test

## What happened
A broken connection string made the integration marker skip all 47 integration tests for two weeks while CI stayed green. A regression shipped to staging unnoticed.

## Why
The skip condition logged nothing and the test summary ("312 passed, 47 skipped") looked normal because unit-test skips were already common.

## When discovered
When a staging smoke test failed on a code path the integration suite was supposed to cover.

## General rule
Treat an unexpected jump in skipped-test count as a failure: pin the expected skip count in CI, or fail the build when integration tests skip for environmental reasons.
